# 코어 JSX 엘리먼트

UI 컴포넌트의 범용성을 위해서 여러가지 유틸 기능이 포함된 전용 JSX 엘리먼트를 사용한다.

## Chakra JSX 엘리먼트

코어 JSX 엘리먼트를 구현하기 위해서 Chakra UI의 코어 시스템 코드([@chakra-ui/system](https://www.npmjs.com/package/@chakra-ui/system))를 참조했다.

컴포넌트를 구현할 때 네이티브 HTML 엘리먼트를 사용하지 않고 전용 JSX 엘리먼트를 사용한다. 기존 HTML 엘리먼트의 모든 속성과 기능을 사용할 수 있고 추가적인 유틸 속성을 추가하여 개발자 경험을 개선할 수 있다.

![](https://vigor-13-temp.s3.ap-northeast-2.amazonaws.com/design_system_component_chaka_ui_system.png)

코어 시스템 엘리먼트를 사용하면 구현된 UI 컴포넌트와 별개돌 커스텀 컴포넌트를 구현하는데 사용할 수 있다.

자세한 내용은 [Chakra UI 공식문서](https://chakra-ui.com/docs/styled-system/chakra-factory#chakra-jsx-elements)를 참조한다.
